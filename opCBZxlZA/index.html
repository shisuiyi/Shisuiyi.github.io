Window - 安装 Tomcat

1. 官方下载地址

https://tomcat.apache.org/download-90.cgi

window 系统选择 zip

2. 配置 JAVA 环境

直接参考这篇博客就好啦：https://www.cnblogs.com/poloyy/p/12744072.html

3. 启动 Tomcat

解压

进入 bin 目录

文本方式打开 catalina.bat

依赖 jdk，因为已经配置了 jdk 的环境变量，所以下面可以直接引用

文本方式打开 setclasspath.bat

jdk 环境安装好则可以直接使用了


启动服务

点击 bin 目录下的 startup.bat

访问127.0.0.1:8080

注意

可以看到全称并没有去配置环境变量哦

4. 配置环境变量

为了启动方便，不需要每次都点 startup.bat ，还是通过命令行的方式来启动

新建系统变量

TOMCAT_HOME = D:\apache-tomcat-9.0.29
CATALINA_HOME = D:\apache-tomcat-9.0.29

修改 path 值

cmd 窗口验证

也可以直接在运行窗口输入 startup
